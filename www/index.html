INTRODUCTION
Project DAPO is a community project which will be using the user's data to enumerate mosquito hotspots in order to effectively prevent the increasing dengue cases in the Philippines.


DATA-DRIVEN
Mosquito sighting and biting cases will be plotted in order to inform users if there is a vast number of mosquito sightings in their area.


CONTRIBUTE
Help by reporting mosquito sightings and biting cases to warn other users.
Click on the:
- if you have seen a mosquito
- if you have been bitten
